# Scaffold: Financial Anomaly Detector

## Folder Structure
```
finance_anomaly_detector/
├── data/
│   └── transactions.csv
├── models/
├── notebooks/
│   └── EDA.ipynb
├── src/

│   ├── detect_anomalies.py
```

## Quick Start
- Add exported bank data to `data/transactions.csv`
- Run preprocessing and anomaly detection
- Launch dashboard with Streamlit
