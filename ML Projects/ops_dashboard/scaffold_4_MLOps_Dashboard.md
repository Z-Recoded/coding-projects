# Scaffold: ML Ops Dashboard

## Folder Structure
```
mlops_dashboard/
├── data/
│   └── experiments.json
├── logs/
├── src/
│   ├── logger.py
│   ├── db_interface.py
│   └── metrics_utils.py
```

## Quick Start
- Define experiments in `experiments.json`
- Log metrics via `logger.py`
- View metrics in `dashboard.py`
